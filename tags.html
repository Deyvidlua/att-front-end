O que é a tag div?

A tag div é uma abreviação para “divisão”. Ela não representa nada em si, mas serve como um container genérico para agrupar e organizar conteúdo em partes específicas de uma página da web.

Basicamente, a div é uma caixa na qual você pode colocar outros elementos HTML que pertencem a um mesmo grupo ou que precisam ser estilizados de forma semelhante.

Exemplo de uso da tag div:

<div class="header-icons"> <i class='bx bxs-cart'></i> <div id="menu"><i class='bx bx-menu'></i></div> </div>

O que é a tag span

A tag span é um container genérico em linha para conteúdo fraseado, mas não representa nada por natureza.

É muito parecido com o elemento div, mas enquanto o div é um elemento de nível de bloco, o span é um elemento em linha.

Exemplo de uso da tag span: <p>Uma <span class="corDeFundoVerde">cor de fundo verde</span> implica perfeitamente a beleza da natureza. </p> <p>Um estilo de fonte em <span id="estilo-fonte">itálico</span> pode ser fundamental para enfatizar um texto.</p>

li

O elemento HTML li (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada (ol), uma lista desordenada (ul) , ou um menu e representa uma única entidade dessa lista.
  1. Primeiro item
  2. Segundo item
  3. Terceiro item
